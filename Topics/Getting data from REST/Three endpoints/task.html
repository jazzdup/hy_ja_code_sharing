This problem is fully supported by version 2021.9 of the EduTools plugin and newer. To avoid possible errors, please make sure that the plugin is updated. Alternatively, you can solve it using the web interface.

To consolidate your understanding of what types of data can be returned from a controller, implement 3 endpoints:

  • GET /value: returns the number 1;
  • GET /text: returns the text two;
  • GET /json: returns the following JSON: {"number": 3}.

Try to use int, String, and Map.

In all mentioned cases the service should respond with 200 (OK) status code.
